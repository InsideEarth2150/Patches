   This package updates The Moon Project to v2.2.1.1B Alpha
   The Update was only tested with Multiplayer/Skirmish.
   For Singleplayer (Campaigns) its better you use only Patch 1.3 or 2.1.1.
   ----------------------------------------------------------------------------------
   ----------------------------------------------------------------------------------
   WARNING
   ----------------------------------------------------------------------------------
   - Incorrect installation can result in abnormal behaviour such as game crashing.
   - This update can be applied to the following version: 2.2.1v2
   ----------------------------------------------------------------------------------
   *** Installation ***
   ----------------------------------------------------------------------------------
   - Ensure The Moon Project is already patched to version 2.2.1v2 (https://discord.com/channels/572336961143177216/856887858316181525/856994949493161995).
   - Put Modules\TheNewMoon.ieo int your game directory \Modules directory (next to EarthExtensions.ieo)
   -- if you have a copy of your game for GameRanger to mitigate Steam issues (the old pre-ProxyLauncher solution), then either upgrade to Proxy launcher (recommended, https://github.com/InsideEarth2150/ProxyLauncher), or apply the changes to your copy installation as well
   - Put WDFiles\Update2211B_AlphaTest.wd
   - The installation process is manual to make it easier to rollback for tournament games
   - Run the game and see if the patch is applied - background images should be changed
   ----------------------------------------------------------------------------------
   *** Deinstallation ***
   ----------------------------------------------------------------------------------
   - Remove WDFiles\Update2211B_AlphaTest.wd from your game directory (renaming it or changing extension won't work)
   - Remove Modules\TheNewMoon.ieo from your game directory (and your game copy for GR if you have it)
   - run the game and see if the patch is sucessfully removed - background images should be changed back to normal
   ----------------------------------------------------------------------------------
   *** Change list ***
   ----------------------------------------------------------------------------------
   ** Bug fixes: **
   ----------------------------------------------------------------------------------
   - disabled heavy weapon subslot for building grabber
   - fixed projectiles passing through UCS walls
   - easier targeting against ED/UCS walls, bridges and trees
   - fixed ED/UCS unupgraded rockets and heavy rockets misfiring
   - fixed heavy sonic subweapon slot
   - fixed misplaced Hydra I weapon slot
   - fixed misplaced Leviathan weapon slot
   - disabled repair module for big ships
   - removed wall construction smoke effect, as it could potentially crash the game
   - passable passives like railroads, opx roads and lights no longer mess up targeting of units/buildings on top of them
   ----------------------------------------------------------------------------------
   ** Balancing Changes: **
   ----------------------------------------------------------------------------------
   * LC - Lunar Corporation *
   ----------------------------------------------------------------------------------
   - changed Guardian building transporter from the fastest (18) to medium (11)
   - increased solar power plant's max panel count from 10 to 12
   ----------------------------------------------------------------------------------
   * ED - Eurasian Dynasty *
   ----------------------------------------------------------------------------------
   - accelerated elevator speed to match that of UCS/LC tunnel entrance
   - fixed Siberia turret rotation issues (a fix that affects balance)
   ----------------------------------------------------------------------------------
   * UCS - United Civilized States *
   ----------------------------------------------------------------------------------
   - no changes
   ----------------------------------------------------------------------------------
   * All Races *
   ----------------------------------------------------------------------------------
   - enhanced MG targeting capabilities against targets above
   - introduced max wall construction range that starts at 1/3 of the map size and then progresses at the same speed as building range
   ----------------------------------------------------------------------------------
   ** Scripts **
   ----------------------------------------------------------------------------------
   - advanced tank script as optional selectable script for ground combat units (changelog in separate file)
   ----------------------------------------------------------------------------------
   ** Map Editor **
   ----------------------------------------------------------------------------------
   - building ruins are now accessible in map editor as low hp (50-300hp) neutral passives
   - single use money artifacts available in map editor
