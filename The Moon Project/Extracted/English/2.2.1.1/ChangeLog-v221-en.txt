   WARNING
   ----------------------------------------------------------------------------------
   - Incorrect installation can result in abnormal behaviour such as game crashing.
   - This update can be applied to the following versions 2.1.0.1, 2.1.9.9 & 2.2
   - This update replaces and remove previous updates (2.1 & 2.2).
   ----------------------------------------------------------------------------------
   Installation
   ----------------------------------------------------------------------------------
   - Ensure The Moon Project is already patched to version 2.1.0.1, 2.1.9.9 or 2.2.
   - Backup any existing installation of EarthTMPExtentions.
   - Backup any remove any existing installation of Mixsets.
   - Backup any existing installation of Autoexec.con.
   - The Installation "should" automatically detect the correct game installation location
   - If it doesn't, ensure you are installing the patch into the correct Game location as defined in (HKLM\SOFTWARE\WOW6432Node\Topware\TheMoonProject\BaseGame\FileSystem "OutputDir")
   - Following the update installation, run the script in the game root called "Finish Update.cmd"
   - Ensure in-game main menu shows v2.2.1
