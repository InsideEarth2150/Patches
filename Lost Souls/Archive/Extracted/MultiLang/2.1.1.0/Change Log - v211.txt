   Installation
   ----------------------------------------------------------------------------------
   - Ensure The Moon Project is already patched to version 2.1.0.1.
   - The Installation "should" automatically detect the correct game installation location
   - If it doesn't, ensure you are installing the patch into the correct Game location as defined in (HKLM\SOFTWARE\WOW6432Node\Reality Pump\LostSouls\BaseGame\FileSystem "OutputDir")
   - Ensure in-game main menu shows v2.1.1
